# Credit Card Fraud Detection

This project uses Machine Learning and ANN to detect fraudulent transactions.

## Features
- Handles imbalanced dataset
- Uses multiple ML models
- ANN model included
- Streamlit UI for prediction

## 🚀 How to Run the Project

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/credit-card-fraud-detection.git
cd credit-card-fraud-detection
```

---

### 2️⃣ Install Required Libraries

```bash
pip install -r requirements.txt
```

---

### 3️⃣ Run the Streamlit App

```bash
streamlit run app.py
```

---

### 4️⃣ Open in Browser

After running the above command, you will see a local URL like:

```
http://localhost:8501
```

Open this link in your browser to view the application.

---

## ⚠️ Important Notes

* Do NOT run using: `python app.py`
* Always use: `streamlit run app.py`
* Make sure all dependencies are installed properly

---

## ✅ Output

* Enter transaction details using the UI
* Click **Predict**
* The app will show whether the transaction is:

  * ⚠️ Fraudulent
  * ✅ Legitimate
